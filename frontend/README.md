# FalconAI Frontend Dashboard

Next.js (App Router) + TypeScript + Tailwind CSS admin dashboard for FalconAI backend.

## Run

1. Install dependencies:

```bash
npm install
```

2. Start development server:

```bash
npm run dev
```

App runs at http://localhost:3000.

## API Base URL

Configured in `services/api.ts`:

- `https://falconaiapi.ebizonstg.com/api/`
